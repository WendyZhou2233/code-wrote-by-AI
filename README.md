# code-wrote-by-AI
一些ai生成的小代码，实现了一些自己的想法

class-anti-pause:
一个轻量级的控制台脚本，用于超星学习通网课页面，实现：
🛡️ 屏蔽失焦、切屏、鼠标移出等行为检测
▶️ 自动点击视频播放按钮（在视频播放区域模拟真实点击）
🔁 视频播放完成后自动跳转下一节，等待约8秒后自动播放
📝 遇到章节测验时自动暂停跳转，等待手动完成

🚀 使用方法
  打开目标网课页面（确保页面已完全加载）
  按 F12 打开开发者工具，切换到 Console（控制台） 标签
  粘贴下代码并回车执行

⚠️ 注意事项
  仅适用于超星学习通，其他平台未测试。且仅在本人的某课程界面进行了测试
  不包含答题功能，遇到章节测验时脚本会暂停自动跳转，请手动答题后点击下一节
  纯粹解决挂机加自动播放下一视频的需求，代码简洁、无外部依赖，适合学习和日常使用
