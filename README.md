# code-wrote-by-AI
一些ai生成的小代码，实现了一些自己的想法

class-anti-pause:
🎯 仅针对 blur、mouseout、mouseleave、visibilitychange、mousemove、mouseover 等检测事件进行拦截
🛡️ 采用捕获阶段阻止事件传播，不会破坏 click 或其他正常交互
🧹 无需刷新即可生效，刷新页面后脚本效果自动消失，无任何永久修改
📦 开箱即用，直接复制到浏览器控制台或注入为书签脚本
🚀 使用方法
  打开目标网课页面（确保页面已完全加载）
  按 F12 打开开发者工具，切换到 Console（控制台） 标签
  粘贴下代码并回车执行
⚠️ 注意事项
  脚本不会自动点击“下一节”或完成视频播放，仅用于辅助你已有的自动流程（如配合定时器点击按钮）
